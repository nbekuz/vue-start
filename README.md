# Vue 3 + Tailwind CSS 4 + i18n + Axios + Ant Design Vue

Bu loyiha Vue 3, Tailwind CSS 4, Vue i18n, Axios va Ant Design Vue bilan yasalgan.

## Qurilish uchun zarur:

- Node.js 16+
- npm yoki yarn

## Setup

```bash
# Dependencies o'rnatish
npm install

# Development server ishga tushurish
npm run dev

# Production uchun build qilish
npm run build

# Preview uchun
npm run preview
```
